Feedback vs นินทา

มี.ค. 2021

พวกเรารู้กันอยู่แล้วว่า feedback เป็นสารอาหารสำคัญที่ทำให้มนุษย์พัฒนา แต่การที่เราจะส่ง feedback ไปให้คนนั้นๆ ก็เป็นเรื่องยากเหลือเกิน เพราะ feedback ที่ดีอย่างเดียวก็ไม่ได้หมายความว่าจะทำให้ผู้รับสารเกิดการเปิดใจ และนำไป improve ต่อยอดได้

ดังนั้นการปั้น feedback ดีๆ อย่างเดียวไม่พอ ต้องมีการดูลาดเลา จังหวะจะโคน บรรยากาศเอยอะไรเอย เพื่อให้พร้อมทั้งผู้ส่งสารและผู้รับสาร โดยรวมแล้วเป็นทักษะที่เราพัฒนาได้อย่างไม่มีที่สิ้นสุด ทุกครั้งที่ผู้เขียนได้เห็นคนใช้ทักษะนี้ได้อย่างเทพ ผู้เขียนจะร้อง “ว้าววววววววววววววว” ในใจดังๆ และหวังว่าจะทำได้บ้าง

ส่วนการนินทา อันนี้ไม่ต้องมีทักษะใดๆ เลยค่ะ แค่ต้องไม่ให้คนที่เราอยากจะนินทาได้ยินเป็นพอ และในเรื่องของการพัฒนา ไม่ต้องพูดถึงค่ะ ตกต่ำทั้งด้านจิตใจของคนนินทา และหากคนที่ถูกนินทาได้ยิน ก็กลายเป็นขยะในใจดีๆ นี่เอง

ผู้เขียนเองไม่ได้เป็นคนที่ไม่เคยนินทานะคะ เป็นคนที่ชอบการนินทามาก ได้ความสบายใจชั่วคราว ได้ระบายอารมณ์ แต่หากมีสติดีครบถ้วน ก็จะประเมินเสมอว่า เราอยากจะ feedback เพื่อ improve เรื่องนี้ หรืออยากจะนินทาสร้างขยะ
