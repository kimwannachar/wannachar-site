รู้จัก HELM กันนะ

เม.ย. 2023

เจ้า helm ถูกสร้างขึ้นในปี 2015 โดย Deis (ที่ต่อมาถูก Microsoft ซื้อไป) และได้ถูกแนะนำให้เป็นที่รู้จักในงานเปิดตัว KubeCon ด้วยชื่อ Helm Classicในเดือน November 2015 ต่อมาในเดือน January 2016 ได้ถูกรวมเข้าไปอยู่ใน Google’s deployment manager for Kubernetes โดยตอนนี้นี่แหละที่ Helm ได้เกิดเป็น Helm Project เป็นตัวเป็นตนของตัวเอง

Helm คือ open source package manager for Kubernetes 
เป็นตัวช่วยในการ deploy ใช้ในการ searching, installation, uninstalling และ upgrading ของใน cluster Kubernetes โดยการ config ค่าผ่านไฟล์ YAML

YAML มากมายไว้เอามาแก้มาเติมกัน
go-chart
├── Chart.yaml
├── charts
├── templates
│ ├── NOTES.txt
│ ├── _helpers.tpl
│ ├── deployment.yaml
│ ├── hpa.yaml
│ ├── ingress.yaml
│ ├── service.yaml
│ ├── serviceaccount.yaml
│ └── tests
│ └── test-connection.yaml
└── values.yaml

Helm deploy อะไรได้บ้าง : Namespace, NetworkPolicy, ResourceQuota, LimitRange, PodSecurityPolicy, PodDisruptionBudget, ServiceAccount, Secret, SecretList, ConfigMap, StorageClass, PersistentVolume, PersistentVolumeClaim, CustomResourceDefinition, ClusterRole, ClusterRoleList, ClusterRoleBinding, ClusterRoleBindingList, Role, RoleList, RoleBinding, RoleBindingList, Service, DaemonSet, Pod, ReplicationController, ReplicaSet, Deployment, HorizontalPodAutoscaler, StatefulSet, Job, CronJob, Ingress และ APIService
